jQuery - AJAX - PHP:

Create JSON response for jQuery

Initialize

PHP: require jQuery library 
  1. require_once 'libraries/jQuery.php';
  2. // many actions
  3. // ...
  4.  
  5. // get JSON response
  6. jQuery::getResponse();
HTML: initialize jQuery and php 
  1. <script src="javascript/jquery.js" type="text/javascript"></script>
  2. <script src="javascript/php.js" type="text/javascript"></script>
JavaScript: call AJAX (example) 
  1. // do an ajax post request (example)
  2. $.ajax({
  3.    // AJAX-specified URL
  4.    url: "http://"+document.domain+"/ajax.php",
  5.    // JSON
  6.    type: "POST",
  7.    data: {},
  8.    dataType : "json",
  9.    // Handle the success event
  10.    // response parser call (it's requried for work php)
  11.    success: function(data, textStatus){
  12.        return php.success(data, textStatus);
  13.    }
  14. })
